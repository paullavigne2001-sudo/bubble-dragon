# 🐉 Bubble Dragon

Bubble Dragon est un jeu rétro inspiré de Bubble Bobble, recréé avec un moteur JavaScript léger et un rendu style Amstrad CPC.

## 🎮 Jouer
👉 https://paullavigne2001-sudo.github.io/bubble-dragon/

## 🎮 Contrôles
- ⬅️ ➡️ : déplacer
- ⬆️ : sauter
- Espace : tirer des bulles
- E : mode éditeur

## 🧩 Éditeur de niveaux
- clic gauche : ajouter plateforme
- clic droit : supprimer
- S : exporter le niveau (JSON console)

## 🎨 Features
- rendu pixel style Amstrad CPC
- bulles physiques (tir horizontal + flottement)
- éditeur intégré
- système de niveaux ASCII

## 🚀 Tech
- HTML5 Canvas
- JavaScript vanilla
- aucun framework
